Question: How to use inventory manager

In inventory manager, you can add, and update any items in inventory.

How to Add inventory?

To add items to the inventory, first, make sure you have fill out all information about the item, this includes image. You can add shoe sizes when adding an item or you can leave it black and add it later.

Once you finish, click Button 'Add new Item'. If everything went well you will be prompted with successful message. And you can see your recently added item on the left list.

How to update inventory?

To update items, first, make sure you have fill out all information about the item.

Once you finish, click Button 'Update Selected Item'. If everything went well you will be prompted with successful message. And you can see your recently added item on the left list.

Answered: In this form you can add, update, or disable users information. Here you can see list of users, you can search any users from yellow search bar above. When you first open this form you can only see customers name, but if you click on the show employee it will show only employees name.

You can see 4 different buttons/options you can choose. Click on button according to your choice.

Question: How to Add new users

Answered: To update user information, first, select the user name from the menu then (make sure the entire row is selected/highlighted in blue). Then click 'Update / View selected user information' button.

Once clicked you will see another form pops up where you can add all users information. Here, you can edit all information as you'd like.
